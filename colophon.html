Colophon

I built this site in HTML5 and CSS3 with Jekyll. The design is based on the version I created with Django in 2013, but I have revised the design of the page headers, refined the responsive elements and gone with a so-hip-it-hurts cards layout for the homepage. Hey, if it's good enough for Facebook, it's good enough for me.

Kit

Hardware

I work on a 2012 Apple Mac Mini with a Quad-core i7 @ 2.5GHz and a 256Gb SSD. I added 16Gb RAM from OWC, and two USB3 MiniStacks to house 5Tb of external storage. Attached is a (sadly discontinued) Razer Blackwidow (Mac Edition) keyboard, because I like mechanical keyboards. They go click. I point at things with a Wacom Bamboo Create (6" x 4") pen tablet my wife got me a couple of years ago.

At work I use a 2014 MacBook Pro 15" with Retina Display, which is gorgeous to use, especially with 16Gb RAM and that PCI SSD!

Software

I spend most of my days in Axure RP 7.0 and Bohemian Coding's Sketch 3. If I have to write code (like for building this site), I do it in Sublime Text, which I don't love, but it's the best there is right now.

I'm waiting on Affinity to release their InDesign killer and then I could happily dump the entire Adobe Creative Cloud. I'd only use it when forced to at work. Between Sublime Text, Sketch, Pages and Pixelmator, I'm getting by pretty well. It would be a nice piece of poetic justice to see a successor to PagePlus take a chunk out of InDesign, just as it was great seeing InDesign, the successor to PageMaker, knock Quark XPress off it's perch ten years ago.

I augment my development tools with CodeKit and Git (specifically that nice GitHub for Mac application) when developing for myself. I have had to use Grunt in the past, but I bought my Mac for a reason, and it wasn't to edit config files.

I'm back using Subversion at work, supplemented with Versions for the Mac. But GitHub is how I roll at home.

As much as I'd like to never see Windows again, we have to test our sites. I use Parallels Dekstop because it came free with my Mac, and virtual machines downloaded from Modern.ie courtesy of Microsoft. I also have to run Visual Studio at work, but I'll be damned if I use it to do anything but run a local copy of the application.

My daily browsing is done in Safari, and Flash is not installed on my system. The superior developer tools in Google Chrome make that my development browser of choice, and its built-in implementation of Flash Player make it my fallback when using sites that require it.
